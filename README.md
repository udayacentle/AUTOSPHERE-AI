# AUTOSPHERE-AI

## Quick start (run the server and web app)

**Terminal 1 – backend:**

```bash
cd autosphere_full_production_monorepo/services/backend
npm install
npm start
```

**Terminal 2 – frontend:**

```bash
cd autosphere-web
npm install
npm run dev
```

- Backend: **http://localhost:3000**
- Web app: **http://localhost:5173**
- Login: **http://localhost:5173/auth/login**
